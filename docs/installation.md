# Installation

## Requirements
- Python 3.9+
- numpy
- pandas
- matplotlib
- mpmath

## Editable Installation

```bash
pip install -e .
```

This installs:

- `reginafield` package
- `reginafield-cli` executable

---

## Verify Installation

```bash
python -c "import reginafield; print(reginafield)"
```
